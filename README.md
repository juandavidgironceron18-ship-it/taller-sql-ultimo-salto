# Taller SQL - El Último Salto

Repositorio con la solución del taller práctico de SQL para la tienda
"El Último Salto".

## Estructura
- ejercicios/: contiene un archivo SQL por cada ejercicio del taller.

## Cómo ejecutar
Los archivos `.sql` pueden ejecutarse directamente contra la base de datos
`tienda.db` usando sqliteonline.
